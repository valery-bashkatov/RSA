The RSA class provides a set of tools for working with RSA cryptographic algorithm.

Keys Generation

    Generates RSA keys with specified key size.

    static open func generateKeyPair(withSize keySize: Int) throws -> (publicKey: SecKey, privateKey: SecKey)

    Parameters

    keySize

    This value defines key size in bits. May have values of 512, 768, 1024 or 2048.

    A tuple with public and private key.

Digital Signature
  • sign(data:using:digest:)

    Signs a data (digest) using private key and returns a digital signature.

    Throws

    An RSAError if an error occurs.

    Declaration

    Swift

    static open func sign(data: Data, using privateKey: SecKey, digest: SecPadding) throws -> Data

    Parameters

    data

    The data to be signed.
    privateKey

    The private key with which to sign the data.
    digest

    The digest, which is used for signing. Available values: PKCS1SHA1, PKCS1SHA224, PKCS1SHA256, PKCS1SHA384 or PKCS1SHA512.

    Return Value

    The digital signature of data.
